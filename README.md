# Traductor de Emojis — Compilador Completo

Compilador desarrollado en Python que traduce texto a emojis mediante un proceso de análisis léxico, sintáctico y semántico.

El sistema analiza oraciones válidas, genera código intermedio, optimiza instrucciones y produce código objeto representado con emojis.

## Fases del Compilador

### 1. Análisis Léxico
Divide el texto en tokens.

### 2. Análisis Sintáctico
Valida estructura de la oración.

### 3. Análisis Semántico
Verifica reglas lógicas.

### 4. Generación de Código
Genera instrucciones intermedias.

### 5. Optimización
Elimina instrucciones repetidas.

### 6. Código Objeto
Traduce a emojis.
